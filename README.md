# AI Resume Studio

## Description
A smart web-based platform to generate resume, portfolio, and analyze resume strength.

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- Chart.js
- ReportLab

## Features
- Resume Generator
- Cover Letter Generator
- Portfolio Builder
- Resume Analysis with Graph
- PDF Download

## Installation

pip install -r requirements.txt
python app.py

## Deployment
Can be deployed on Render or Railway.
